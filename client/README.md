# Emotional Intelligence Pipeline Frontend

This folder contains the frontend for the emotional intelligence pipeline.

## Included

- `src/` contains the main React + TypeScript app used by this repository.
- `prototype/` contains the imported original CalmDownAI static prototype.

## Imported Prototype Files

The following files were brought into `client/prototype/`:

- `app.js`
- `index.html`
- `README.md`
- `styles.css`

## Run The Frontend

From the `client` folder:

```bash
npm install
npm run dev
```

## Notes

- The React app in `src/` is the primary implementation.
- The static prototype in `prototype/` is preserved as a standalone reference version.
